# Delay-NIH-plug
A simple delay effect plugin built with [NIH-plug](https://github.com/robbert-vdh/nih-plug) plugin framework.

## Build
```bash
cargo install --git https://github.com/robbert-vdh/nih-plug.git cargo-nih-plug
git clone https://codeberg.org/lzj15/delay-nih-plug.git
cd delay-nih-plug
cargo nih-plug bundle delay-nih-plug --release
